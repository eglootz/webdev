Using python and the HUE-API to control my LED-lights

February 01, 2022

The reason behind all of this

The project week

We have this special week at our school - everyone has to think of a cool project that interests them and talk with a teacher about it. If they agree, well then everything is settled. This week will be in March, and in it everyone gets seven days to work on the project, write a huge paper about the theory and submit a practical part.

My project

I chose the project you've already seen in the title - I wanted to control the lights I have in my room. Using the standard program would be way too easy, so I wanted to learn how to work with API's, control the lights using one, put everything in a pretty GUI and code some special features. One of those is letting my lights turn red whenever someone comes near my door.

The plan

Things I want to achieve:

  • a practical program to control my lights in more ways than before
  • a pretty graphical user interface
  • a way to detect motion
  • syncing my pc-lights and those in my room

The language (it's Python)

I plan on using python, because that's the only language I'm comfortable enough to take on a fairly big project like this. I may have to learn JavaScript in the process, because of the work with API's.
